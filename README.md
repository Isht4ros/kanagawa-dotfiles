# kanagawa-dotfiles
Kanagawa-themed Windows dotfiles

### Used

- [GlazeWM](https://github.com/glzr-io/glazewm) — Tiling wm
- [YASB](https://yasb.dev/) — Status bar
- [Yazi](https://github.com/sxyazi/yazi) — Terminal file manager
- Windows Terminal — Terminal
- [Cava](https://github.com/karlstav/cava) - Audio Visualizer
- [Fastfetch](https://github.com/fastfetch-cli/fastfetch) — System info fetch
- [Flow Launcher](https://github.com/Flow-Launcher/Flow.Launcher) — Spotlight for Windows

### Install
› Windows theme:
1. Create a system restore point
2. Install and run [UltraUXThemePatcher](https://mhoefs.eu/software_uxtheme.php?lang=en) — this patches Windows to allow custom themes
3. Restart your PC
4. Download and apply the [Kanagawa for Win11](https://www.deviantart.com/niivu/art/Kanagawa-for-Windows-11-1021605687) theme by niivu


› Firefox custom css:
1. Go to `about:config` in Firefox and set `toolkit.legacyUserProfileCustomizations.stylesheets` to `true`
2. Go to `about:support` and click **Open Profile Folder**
3. Create a `chrome` folder inside it
4. Place `userChrome.css` inside the `chrome` folder
5. Restart Firefox
